# Activity tracker

Demo Vuejs3 composition api app.

[Click heer to see demo.]()
## requirements
- nodejs v^20

## Project Setup

```sh
npm i
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
